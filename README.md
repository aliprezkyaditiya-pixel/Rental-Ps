# Rental PS Website

A professional PlayStation rental website with admin dashboard.

## Features
- ✅ Booking system with QRIS & Cash payment
- ✅ Admin dashboard for validation
- ✅ Real-time transaction updates

## Tech Stack
- HTML, CSS, JavaScript
- Firebase (Firestore, Auth)
- Vercel Deployment

## Setup
1. Clone repository
2. Configure Firebase
3. Deploy to Vercel
